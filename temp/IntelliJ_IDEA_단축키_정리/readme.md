# IntelliJ IDEA 단축키 정리

`⌘` `⌥` `⌃` `⇧`


- `cmd` + `T` : Merge
- `cmd` + `P` : 파라미터 보기
- `ctrl` + `T` : 리팩토링 관련
- `ctrl` + `R` : 이전 실행 다시 실행


- `cmd` + `shift` + `T`: 테스트 코드 생성

- `cmd` + `opt` + `M`: 함수 추출
- `cmd` + `opt` + `V`: Introduce Variable

- `Shift` + `F6`: 변수명 변경
- `Shift` + `F10`: 
- `cmd` + `,` : Preferences
- `cmd` + `;` : Project Structure
