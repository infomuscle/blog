# IntelliJ IDEA 단축키 정리

`⌘` `⌥` `⌃` `⇧`


- `cmd` + `T` : Merge
- `cmd` + `P` : 파라미터 보기
- `cmd` + `,` : Preferences
- `cmd` + `;` : Project Structure

- `ctrl` + `T` : 리팩토링 관련
- `ctrl` + `R` : 이전 실행 다시 실행

- `cmd` + `shift` + `T`: 테스트 코드 생성

- `ctrl` + `opt` + `O`: 불필요한 import 제거

- `cmd` + `opt` + `M`: 함수 추출
- `cmd` + `opt` + `V`: Introduce Variable
- `cmd` + `opt` + `L`: 코드 포맷
- `cmd` + `opt` + `N`: 인라인

- `Shift` + `F6`: 변수명 변경
